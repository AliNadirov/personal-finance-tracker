# Personal Finance Tracker

A modern web application for managing personal finances, tracking income and expenses, and monitoring budget through a structured dashboard.

Live Demo: https://personal-finance-tracker-1-project.netlify.app/  
Repository: https://github.com/AliNadirov/personal-finance-tracker  

---

## Overview

This project was developed as a front-end application focused on building a financial dashboard experience.

I was primarily responsible for the front-end architecture, UI implementation, and core features.

---

## Features

- Add and delete income and expense entries  
- Categorize transactions (Food, Transport, Health, etc.)  
- Dynamic calculation of total income, expenses, and balance  
- Dashboard view with financial summaries  
- User authentication using localStorage  
- Predefined mock users for testing  
- OAuth login UI (Google & Facebook – simulated)  
- Profile / Settings menu with logout functionality  
- Responsive design for desktop and mobile  
- Persistent data using localStorage  

---

## Tech Stack

Frontend: React, Vite  
Styling: CSS3  
State & Storage: LocalStorage  
Version Control: Git & GitHub  

---

## Project Status

This application is a front-end prototype and includes simulated features:

- Uses mock data instead of a real backend  
- Authentication handled via localStorage  
- OAuth login is UI-only (not fully integrated)  
- Financial data is manually entered by users  
- Some visual elements (charts) are static placeholders  

---

## What This Project Demonstrates

- Structuring a scalable React application  
- Managing state and user interactions  
- Creating reusable UI components  
- Designing responsive layouts  
- Simulating real-world product features without backend

## Getting Started

Clone the repository and run locally:

```bash
git clone https://github.com/AliNadirov/personal-finance-tracker.git
cd personal-finance-tracker
npm install
npm run dev
```
